🔧 Fix: Bug Partecipanti e Altre Spese

❌ Problema Riscontrato

Quando si selezionavano "altre spese", la lista dei partecipanti si aggiornava in modo anomalo e faceva sbagliare i conti del costo per partecipante.

Causa del problema:

  • L'auto-selezione del checkbox triggerava eventi change ricorsivi
  • I listener degli input non distinguevano tra checkbox e altri campi
  • Questo causava chiamate multiple a updateTotals()
  • La lista dei partecipanti veniva contata più volte in modo errato

✅ Soluzione Implementata

1. Separazione dei listener

Le checkbox ora hanno listener dedicati separati dagli altri input (price, name, note)

if (input.type === 'checkbox') { ... return; }

2. Controllo stato checkbox

Prima di impostare checked = true, verifichiamo che non sia già selezionata

if (checkbox && !checkbox.checked && ...)

3. Prevenzione eventi ricorsivi

Le checkbox vengono modificate programmaticamente senza triggerare eventi aggiuntivi

🧪 Come Testare il Fix

  1. Apri la pagina principale dell'applicazione
  2. Vai alla sezione "Preventivi"
  3. Aggiungi 3-4 partecipanti (es. Marco, Luca, Sara, Anna)
  4. Verifica il numero di partecipanti mostrato nella lista
  5. Inserisci un prezzo in "Altre spese" (es. 100€)
  6. Verifica che:
    • ✅ Il checkbox si seleziona automaticamente
    • ✅ Il numero di partecipanti rimane invariato
    • ✅ Il totale si aggiorna correttamente
    • ✅ Il costo per partecipante è calcolato correttamente (totale / numero partecipanti)
  7. Aggiungi altre spese e verifica che il comportamento sia consistente
  8. Deseleziona e riseleziona le checkbox manualmente per verificare che tutto funzioni

📊 Esempio di Test

Scenario di test:

  • Partecipanti: 4 persone (Marco, Luca, Sara, Anna)
  • Trasporto: 400€
  • Alloggio: 800€
  • Vitto: 300€
  • Altre spese 1: 100€ (inserita senza selezionare checkbox)
  • Altre spese 2: 50€ (inserita senza selezionare checkbox)

Risultato atteso:

  • ✅ Entrambe le checkbox si selezionano automaticamente
  • ✅ Numero partecipanti rimane: 4
  • ✅ Totale: 1.650€
  • ✅ Costo per partecipante: 412,50€
