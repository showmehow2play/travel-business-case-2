📦 Esporta Dati Locali → sample-data.js

Questo strumento esporta i tuoi dati salvati nel localStorage locale in un formato compatibile con sample-data.js, così potrai includerli nell'app pubblicata su GitHub.

ℹ️ Come Funziona

I tuoi dati sono salvati nel localStorage del browser. Questo tool li legge e genera il codice JavaScript da copiare nel file js/sample-data.js.

📝 Prossimi Passi

  1. Clicca su "Genera Codice" per creare il codice JavaScript
  2. Clicca su "Copia negli Appunti" per copiare il codice
  3. Apri il file js/sample-data.js nel tuo editor
  4. Sostituisci tutto il contenuto con il codice copiato
  5. Salva il file
  6. Pubblica su GitHub - i tuoi dati saranno inclusi! 🚀
