# 🚀 Guida Rapida - Business Case Viaggi

## Primi Passi (5 minuti)

### 1️⃣ Apri l'Applicazione
- Fai doppio clic su `index.html`
- L'app si aprirà nel tuo browser predefinito
- Non serve installazione!

### 2️⃣ Crea il Tuo Primo Scenario

**Passo 1: Clicca "+ Nuovo Scenario"**

**Passo 2: Compila i Dati Base**
```
Nome Scenario: Viaggio Roma 2026
Destinazione: Roma, Italia
Data Inizio: 01/06/2026
Data Fine: 07/06/2026
```

**Passo 3: Aggiungi Partecipanti**
- Scrivi "Mario Rossi" → Clicca "+ Aggiungi"
- Scrivi "Luigi Verdi" → Clicca "+ Aggiungi"

**Passo 4: Cerca Voli (Opzionale ma Consigliato!)**
- Clicca "✈️ Cerca voli" sotto il campo Trasporto
- Inserisci città di partenza (es: Milano)
- Le altre info sono già pre-compilate!
- Clicca su Skyscanner, Google Flights, o altri siti
- Confronta i prezzi e trova il volo migliore

**Passo 5: Inserisci le Spese**
```
🚗 Trasporto:    € 500  (usa il prezzo del volo trovato!)
🏨 Alloggio:     € 800
🍽️ Vitto:        € 400
🚙 Auto:         € 200
🎭 Attività:     € 150
```

**Passo 6: Salva**
- Clicca "💾 Salva Scenario"
- Vedrai i totali calcolati automaticamente!

### 3️⃣ Usa la Ricerca Voli Integrata! ✈️

**Perché è utile:**
- ✅ Pre-compila automaticamente destinazione e date
- ✅ Apre 5 siti di comparazione in un click
- ✅ Confronta centinaia di compagnie aeree
- ✅ Trova i prezzi migliori rapidamente

**Come funziona:**
1. Clicca "✈️ Cerca voli" nel form
2. Verifica/modifica città di partenza
3. Clicca su uno o più siti (si aprono in nuove tab)
4. Confronta i prezzi
5. Inserisci il costo nel campo Trasporto

### 4️⃣ Crea Altri Scenari per Confrontare

Ripeti i passi sopra con destinazioni o configurazioni diverse:
- Scenario 2: Milano
- Scenario 3: Firenze

### 5️⃣ Confronta gli Scenari

1. Vai su "🔄 Confronta"
2. Seleziona 2 o più scenari
3. Clicca "Confronta Selezionati"
4. Analizza tabelle e grafici!

## 💡 Trucchi Veloci

### Ricerca Voli Veloce
- **Pre-compilazione automatica** = Risparmia tempo
- **Confronta sempre 2-3 siti** = Trova il prezzo migliore
- **Modalità incognito** = Evita aumenti di prezzo
- **Prenota in anticipo** = Prezzi più bassi

### Scorciatoie
- **Invio** nel campo partecipante = Aggiungi automaticamente
- **Duplica** uno scenario per creare varianti rapidamente
- **Note** per ricordare dettagli importanti

### Calcoli Automatici
- ✅ Totale spese aggiornato in tempo reale
- ✅ Costo per persona calcolato automaticamente
- ✅ Durata viaggio in giorni

### Esportazione Rapida
1. Clicca "📤 Esporta"
2. Scegli formato:
   - **JSON** → Backup
   - **Excel** → Analisi
   - **PDF** → Presentazione

## 🎯 Casi d'Uso Comuni

### Caso 1: Confronto Destinazioni
```
Scenario A: Roma (€2,050)
Scenario B: Milano (€1,800)
Scenario C: Firenze (€1,950)

→ Milano è la più economica!
```

### Caso 2: Variazioni Partecipanti
```
Scenario A: 2 persone (€1,025/persona)
Scenario B: 4 persone (€512/persona)

→ Più partecipanti = Costo per persona ridotto!
```

### Caso 3: Budget Planning
```
Budget disponibile: €2,000
Scenario creato: €2,050

→ Riduci alcune spese o cerca alternative!
```

## ⚡ Risoluzione Problemi Rapida

| Problema | Soluzione |
|----------|-----------|
| Dati non salvati | Controlla che localStorage sia abilitato |
| Non si apre | Usa un browser moderno (Chrome, Firefox, Safari) |
| Export non funziona | Abilita i popup nel browser |
| Importazione fallita | Usa solo file JSON esportati dall'app |
