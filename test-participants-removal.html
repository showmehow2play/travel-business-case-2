🧪 Test Rimozione Partecipanti

Verifica che i pulsanti ✕ funzionino correttamente

📋 Istruzioni Test

  1. Clicca sul pulsante ✕ accanto a un partecipante
  2. Il partecipante dovrebbe essere rimosso immediatamente
  3. Se rimuovi tutti, dovrebbe apparire il messaggio "Nessun partecipante"
  4. Aggiungi nuovi partecipanti usando il campo sotto

📋 Partecipanti Aggiunti

Nessun partecipante aggiunto. Aggiungi qualcuno usando il campo sotto.
